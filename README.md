# web_scraper

Python script scraper

For program help use:
         -h or --help

Feed data file use:
        -f or --feed

Get prices NOW use:
        -n or --now

Get CHEAPEST price ever use:
        -c or --cheapest

Get LATEST data feed date use:
        -l or --atest-fetch

Get total AMOUNT of feeds use:
        -a or --fetch-amounts
